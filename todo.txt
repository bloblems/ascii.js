- draw_layer(layer[, x, y, w, h])
	-> Draw only rect from layer

- draw() updates only the displayed the <span> elements (optimization)

- create_style_layer()
- set_style()

- iter_func(func[, layer])
- map_func(func[, layer])
	-> with func(x, y, layer)
	-> Loop through each cell of the layer / canvas and run the passed function
	-> for map_func(), the function should return a character that will update
	   the layer. Technically, the function will fill a new layer to not affect
	   the passed layer and only after the fill, the new layer will be drawn on
	   the passed one.

- polygon(char_edges, char_fill, x1, y1, x2, y2, ...)
- polygon(char_edges, char_fill, [x1, y1, x2, y2, ...])
	-> Can pass coordinates as "unlimited" number of parameters or as an array
- rpolygon(x, y, rx, ry, vertices, char_edges, char_fill = null, angle = 0)
	-> rpolygon for regular polygon: It would replace the shape() function
	-> Can give a fill character, if not provided, no fill
	-> Fill: use a cache layer to draw the shape, fill it and draw the layer
	-> shape() would create an array of points and call polygon()
